# Amazon Product Reviews Sentiment Analysis

## Project Overview

This project focuses on analyzing customer reviews from Amazon products using Natural Language Processing (NLP) techniques. The goal is to determine whether customer reviews express positive, negative, or neutral sentiment.

## Objective

* Analyze customer opinions from product reviews.
* Classify reviews into Positive, Negative, and Neutral categories.
* Understand customer satisfaction levels.
* Visualize sentiment distribution using charts.

## Dataset

The dataset contains Amazon product reviews collected from customers.

### Features Used

* Review Text
* Product Information
* Customer Feedback

## Tools and Libraries

* Python
* Pandas
* TextBlob
* Matplotlib
* Jupyter Notebook

## Methodology

### 1. Data Loading

The dataset was loaded using Pandas.

### 2. Data Exploration

Dataset structure, columns, and missing values were examined.

### 3. Sentiment Analysis

TextBlob was used to calculate sentiment polarity for each review.

Sentiment Categories:

* Positive
* Negative
* Neutral

### 4. Data Visualization

The sentiment distribution was visualized using:

* Bar Chart
* Pie Chart

## Key Insights

* Most customer reviews were positive.
* Negative reviews represented a smaller portion of total reviews.
* Overall customer sentiment indicated high satisfaction with the products.
* Positive customer experiences can contribute to stronger product reputation and increased sales.

## Conclusion

The sentiment analysis revealed that the majority of customers expressed positive opinions about the products. This demonstrates how Natural Language Processing can be used to understand customer feedback and support data-driven business decisions.
